Snaffu
======

Latest Snaffu SDK

To use 0.0.6 SnaffuKit from cocoapods, add this line in your Podfile:

```
  pod 'SnaffuKit'
```

Starting out

In your AppDelegate.m import SnaffuKit and call the startUpToken method with your app token.

```
#import "SnaffuKit.h"
[SnaffuKit startUpToken:@"YOUR_APP_TOKEN"];
```
